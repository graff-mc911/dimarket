## 🔧 Технічні деталі

### Архітектура:
- **Контекст** - `AppContext` зберігає вибрану мову
- **LocalStorage** - Вибір зберігається між сесіями
- **Функція `t()`** - Перекладає кожен ключ
- **Fallback** - Якщо переклад відсутній → English

### Приклад коду:
```typescript
// Використання в компонентах
const { t } = useApp()

// Переклад тексту
<h1>{t('home.heroTitle')}</h1>
// English: "Find Trusted Professionals"
// Українська: "Знайдіть перевірених майстрів"
// Polski: "Znajdź zaufanych fachowców"
```

### Файл перекладів:
`/src/lib/i18n.ts` - містить всі переклади

## 🌟 Основні переваги системи:

### 1. Миттєва зміна
Без перезавантаження сторінки - React автоматично оновлює всі компоненти

### 2. Збереження вибору
LocalStorage запам'ятовує вашу мову назавжди

### 3. Легке розширення
Додати нову мову = додати об'єкт у файл перекладів

### 4. Type-Safe
TypeScript перевіряє, що всі ключі існують

### 5. Fallback
Якщо переклад відсутній → показується англійська версія

## 💡 Корисні команди:

### Додати нову мову:
1. Відкрити `/src/lib/i18n.ts`
2. Додати новий об'єкт з кодом мови
3. Перекласти всі ключі
4. Готово!

### Додати нову фразу:
1. Додати ключ в `translations.en`
2. Додати переклади в інших мовах
3. Використати `t('новий.ключ')` в компоненті
